Title: CDC and Standard CBC/Lab Reference Context (Trusted Summary)
Source: CDC public health references and common clinical laboratory reference standards

Common adult reference context:
- Hemoglobin: approximately 12-16 g/dL (contextual by sex and lab)
- WBC: approximately 4.0-11.0 x10^3/uL
- Platelets: approximately 150-450 x10^3/uL
- Fasting glucose common target context: about 70-99 mg/dL

Interpretation principles:
- Single test anomalies should be rechecked before long-term conclusions.
- Trends across repeat reports improve confidence.
